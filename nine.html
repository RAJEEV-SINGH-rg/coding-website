Table in HTML


✦Basic Table Tags
To create a table in HTML, <table> tag is used
  • <th> tag is used to create table heading
  • <tr> tag is used to create row of the table
  • <td> tag is used to assign data to the cell
✦Example Code and Result
<table>
<tr>
<th>Company</th>
<th>Contact</th>
<th>Country</th>
</tr>
<tr>
<td>Alfreds Futterkiste</td>
<td>Maria Anders</td>
<td>Germany</td>
</tr>
<tr>
<td>Centro comercial Moctezuma</td>
<td>Francisco Chang</td>
<td>Mexico</td>
</tr>
</table>
