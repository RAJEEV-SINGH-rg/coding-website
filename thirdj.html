Java Basic Syntax

Java is a statically-typed, object-oriented programming language with a simple and consistent syntax. Understanding its basic syntax is crucial for writing efficient code. Below are some of the fundamental elements that make up Java syntax.

In Java, every application must have at least one class, and every class contains a `main` method where the program execution begins. A basic Java program looks like this:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

In this example, `public class HelloWorld` defines the class, and `public static void main(String[] args)` defines the main method. The `System.out.println()` function is used to print output to the console.

Java is a strongly typed language, meaning that every variable must be declared with a specific data type. Some common data types in Java include:

  • int: Used for integer values (e.g., 5, -10, 200).
  • double: Used for decimal numbers (e.g., 3.14, -0.001).
  • boolean: Used for true or false values.
  • String: Used for textual data, such as "Hello".

Java also uses operators such as `+`, `-`, `*`, `/` for basic arithmetic and logical operators like `&&`, `||`, and `!` for conditional expressions. Java also supports control flow statements like `if`, `else`, and loops such as `for`, `while`, and `do-while` to control the flow of execution.

For example, an `if` statement checks if a condition is true and executes code accordingly:

int number = 5;
if (number > 0) {
    System.out.println("The number is positive.");
} else {
    System.out.println("The number is non-positive.");
}

This basic syntax is just the beginning, but it forms the foundation for all Java programs.
