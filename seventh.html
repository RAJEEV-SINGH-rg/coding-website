Links


✦To insert Links use < a href="url"> Link text </a>

✦1. href attribute indicates link destination.
target attribute specifies where the link will open, by default it is _self which means the link will open in same window.
It can have following values

  • _self: Default.opens link in same window
  • _blank: opens link in another tab/window
  • _parent: opens link in parent frame
  • _top: opens link in full body of window
