Data Types in Java

In Java, data types specify the type of value a variable can hold. Java is a strongly typed language, meaning every variable must have a data type. There are two main categories of data types: primitive and reference types.

1. Primitive Data Types

Primitive data types are the most basic type of data available in Java. They are not objects and hold simple values directly. Java has 8 primitive data types:

  • byte: 8-bit signed integer
  • short: 16-bit signed integer
  • int: 32-bit signed integer
  • long: 64-bit signed integer
  • float: 32-bit floating-point number
  • double: 64-bit floating-point number
  • char: 16-bit Unicode character
  • boolean: Represents one of two values: true or false

2. Reference Data Types

Reference data types are more complex than primitive data types. They store references to objects rather than the actual values. They can represent arrays, objects, and interfaces. A reference data type can be assigned to any object or array type.

3. Default Values

In Java, every variable has a default value if not explicitly initialized. Here are the default values for each data type:

  • byte: 0
  • short: 0
  • int: 0
  • long: 0L
  • float: 0.0f
  • double: 0.0d
  • char: '\u0000' (null character)
  • boolean: false
  • Reference types: null

4. Type Casting

Type casting in Java refers to converting one data type into another. There are two types of casting:

  • Implicit Casting (Widening): Happens automatically when converting from a smaller data type to a larger one (e.g., from int to long).
  • Explicit Casting (Narrowing): Requires manual conversion when going from a larger data type to a smaller one (e.g., from double to int).

5. Wrapper Classes

Java provides wrapper classes for each of the primitive data types. These classes allow for working with primitives as objects. Here are the wrapper classes:

  • Byte (for byte)
  • Short (for short)
  • Integer (for int)
  • Long (for long)
  • Float (for float)
  • Double (for double)
  • Character (for char)
  • Boolean (for boolean)
