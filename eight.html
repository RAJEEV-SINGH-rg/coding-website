Images in HTML

✦<img> tag is used to embed an image
✦src gives destinantion of image,
✦alt gives alternate name for the image

✦height and width specify dimension of image. Always specify them.

It is always recommended to use style attribute to scale images so that style sheets can not override it.

Use float property to let the image float to left or right
