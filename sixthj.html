Operators in Java

In Java, operators are special symbols used to perform operations on variables and values. Java provides a rich set of operators to perform various tasks like arithmetic, comparison, logical operations, and more.

1. Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations like addition, subtraction, multiplication, and division.

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder of division)

Example:

int a = 10, b = 5;
int sum = a + b; // sum = 15

2. Relational Operators

Relational operators are used to compare two values. These operators return a boolean value (true or false).

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Example:

int a = 10, b = 5;
boolean result = a > b; // result = true

3. Logical Operators

Logical operators are used to combine multiple boolean expressions. They return a boolean value.

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

Example:

boolean a = true, b = false;
boolean result = a && b; // result = false

4. Assignment Operators

Assignment operators are used to assign values to variables. The most common assignment operator is the = operator, but there are others that combine assignment with arithmetic operations.

  • =: Simple assignment
  • +=: Add and assign
  • -=: Subtract and assign
  • *=: Multiply and assign
  • /=: Divide and assign
  • %=: Modulus and assign

Example:

int a = 5;
a += 3; // a = 8

5. Unary Operators

Unary operators are used to perform operations on a single operand. These include increment, decrement, and negation.

  • ++: Increment (increases the value by 1)
  • --: Decrement (decreases the value by 1)
  • +: Unary plus (indicates positive value)
  • -: Unary minus (negates the value)
  • !: Logical NOT (inverts the value of a boolean expression)

Example:

int a = 5;
a++; // a = 6

6. Ternary Operator

The ternary operator is a shorthand for an if-else statement. It is used to evaluate a condition and return one of two values depending on whether the condition is true or false.

condition ? value_if_true : value_if_false;

Example:

int a = 10, b = 5;
int result = (a > b) ? a : b; // result = 10

7. Bitwise Operators

Bitwise operators are used to perform operations on the individual bits of integer values.

  • &: Bitwise AND
  • |: Bitwise OR
  • ^: Bitwise XOR
  • ~: Bitwise NOT
  • <<: Left shift
  • >>: Right shift
  • >>>: Unsigned right shift

Example:

int a = 5, b = 3;
int result = a & b; // result = 1

8. Instanceof Operator

The instanceof operator is used to check whether an object is an instance of a specific class or subclass.

object instanceof ClassName;

Example:

String str = "Hello";
boolean result = str instanceof String; // result = true

9. Conclusion

In summary, Java provides a wide variety of operators for performing different tasks. Whether you are doing basic arithmetic, logical comparisons, or working with bits, understanding these operators is essential for writing efficient Java code. By mastering operators, you can greatly improve your programming skills in Java.
