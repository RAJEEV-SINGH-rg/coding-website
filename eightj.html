Iterative Statements in Java

In Java, iterative statements (also known as loops) are used to execute a block of code repeatedly based on a condition. These loops help in automating repetitive tasks and handling repetitive data operations efficiently.

1. For Loop

The for loop is used when you know in advance how many times the loop should run. It consists of three parts: initialization, condition, and increment/decrement.

Syntax:

for (initialization; condition; increment/decrement) {
  // code to be executed
}

Example:

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

This will print the numbers 0 to 4.

2. While Loop

The while loop is used when you don't know beforehand how many times the loop will run. It continues as long as the specified condition is true.

Syntax:

while (condition) {
  // code to be executed
}

Example:

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

This will also print the numbers 0 to 4.

3. Do-While Loop

The do-while loop is similar to the while loop, but with a key difference: the condition is checked after the loop body is executed. This ensures that the code inside the loop is executed at least once, regardless of whether the condition is true or not.

Syntax:

do {
  // code to be executed
} while (condition);

Example:

int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

This will also print the numbers 0 to 4.

4. For-Each Loop

The for-each loop is used to iterate over elements in arrays or collections. It is a simpler and cleaner way to iterate through the elements without needing an index variable.

Syntax:

for (type element : array) {
  // code to be executed for each element
}

Example:

int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println(number);
}

This will print each element in the array: 1, 2, 3, 4, 5.

5. Break Statement

The break statement is used to exit a loop prematurely when a specific condition is met. It is commonly used to stop a loop when further iterations are unnecessary.

Example:

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // exits the loop when i is 5
    }
    System.out.println(i);
}

This will print the numbers 0 to 4 and then break the loop when i reaches 5.

6. Continue Statement

The continue statement is used to skip the current iteration of a loop and move to the next iteration. It is often used when you want to skip certain conditions inside a loop but continue with the rest of the iterations.

Example:

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue; // skips the iteration when i is 5
    }
    System.out.println(i);
}

This will print the numbers 0 to 9, but skip 5.

7. Nested Loops

In Java, you can have loops inside other loops. These are called nested loops. They are useful when dealing with multi-dimensional data structures, such as matrices or arrays of arrays.

Example:

for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        System.out.println("i: " + i + ", j: " + j);
    }
}

This will print pairs of i and j from 0 to 2.

8. Conclusion

Iterative statements are crucial in programming when you need to repeat tasks or iterate over collections of data. Java provides different types of loops, including the for loop, while loop, do-while loop, and the for-each loop. By mastering these iterative structures, you can write more efficient and concise programs.
